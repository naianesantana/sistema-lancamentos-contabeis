# Sistema de Lançamentos Contábeis (em andamento)
Esse sistema simula, de forma simplificada e direta, um diário contábil, possibilitando o registro de entradas e saídas classificados
por conta.

Ele possui um menu com três funcionalidades: Lançamentos, Contas e Exit. 
As duas primeiras funcionalidades permitem a criação e visualização de seus correspondentes.

Esse projeto nasceu do interesse em aplicar os estudos em Lógica de Prgramação e Algoritmos e Banco de Dados (estudados na faculdade)
ao contexto da empresa que trabalho atualmente, pensando em entender mais afundo os processos e, possivelmente, encontrar soluções
que os otimizem.

O sistema é baseado na linguagem Python e SQL, possuindo conexão com um banco de dados presente no MySQL Workbench.
